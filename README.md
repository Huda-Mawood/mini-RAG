# mini-RAG
this a minimal implementation of the RAG Model for question answering 

## requirements
- python 3.13.11 or later 

### Install Python Using MiniConda
1) Download  and Install miniconda from [here](https://www.anaconda.com/docs/getting-started/miniconda/install)
2) Create a new environment using the following command:
```bash
    $conda creat -n mini-rag-app python=3.13.11
```
3) Activate the environment
```bash
    $conda activate mini-rag-app
```
## Installation
### Install the required packages
```bash
$pip install -r requirements.txt
```
### setup the environments variables
```bash
$ cp .env.example .env
```


set your environment variables in the `.env` file. like `OPENAI_API_KEY` value
